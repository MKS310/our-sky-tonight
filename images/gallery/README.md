# Astronomy Image Gallery

This directory contains astronomy photos that are randomly displayed on the Our Sky Tonight homepage.

## Adding Images

1. Add your astronomy photos to this directory
2. Supported formats: `.jpg`, `.jpeg`, `.png`, `.gif`
3. Recommended image dimensions: 1920x1080 or similar aspect ratio
4. Optimize images for web (compress to reduce file size)

## Current Images

Replace the placeholder images with your own astronomy photos:
- Photos of the night sky over Neenah, WI
- Moon and planet photography
- Deep sky objects (nebulae, galaxies, star clusters)
- Milky Way shots
- Sunset/sunrise with astronomical elements

One image will be randomly selected each time the site rebuilds (daily at 7 AM UTC).
